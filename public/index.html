Welcome to the US States API

This is a RESTful API providing data about the 50 U.S. states. You can retrieve information such as each state's capital, nickname, population, admission date to the Union, and random fun facts (when available).

All API endpoints are under the base path /states. For example:

  • /states – Get data for all states.
  • /states?contig=true – Get data for all contiguous states (the lower 48).
  • /states?contig=false – Get data for the non-contiguous states (Alaska and Hawaii).
  • /states/KS – Get full details for Kansas.
  • /states/KS/capital – Get the capital of Kansas.
  • /states/KS/funfact – Get a random fun fact about Kansas.

You can also add or modify "fun facts" about specific states using POST, PATCH, and DELETE requests to the appropriate /states/:state/funfact endpoints.

See the project documentation for full details on each endpoint. Enjoy exploring the states!
